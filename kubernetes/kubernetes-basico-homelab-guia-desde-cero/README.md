# Kubernetes Básico para Homelab - Ejemplos Prácticos

Ejemplos prácticos para aprender Kubernetes desde cero, comparando conceptos con Docker.

## Estructura

- `01-nginx-basico/` - Primer despliegue: Nginx con Deployment + Service
- `02-nginx-con-volumen/` - Nginx con almacenamiento persistente (PV/PVC)
- `03-nginx-con-ingress/` - Nginx expuesto con Ingress (Traefik)
- `04-wordpress-completo/` - WordPress + MySQL completo (Deployment + Service + PVC + Ingress)
- `docker-compose-equivalente/` - Comparación con Docker Compose

## Requisitos

- K3s instalado (ver guía completa)
- kubectl configurado
- Conocimientos básicos de Docker

## Uso

Cada carpeta contiene ejemplos listos para copiar y pegar. Ver README.md en cada carpeta para instrucciones específicas.
