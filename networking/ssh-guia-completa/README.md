# SSH - Guía Completa para Principiantes

Ejemplos prácticos y scripts para aprender SSH desde cero.

## Contenido

- **scripts/**: Scripts automatizados para tareas comunes
- **examples/**: Ejemplos de comandos y configuraciones
- **ssh_config_example**: Archivo de configuración SSH de ejemplo

## Uso Rápido

### Generar Clave SSH

```bash
./scripts/generate_key.sh
```

### Copiar Clave al Servidor

```bash
./scripts/copy_key.sh usuario@servidor
```

### Probar Conexión

```bash
./scripts/test_connection.sh servidor
```

### Backup Remoto

```bash
./scripts/backup_remote.sh usuario@servidor /ruta/local /ruta/remota
```
