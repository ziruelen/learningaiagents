# AutoKitteh Workflows Python Temporal 2026

Ejemplos base para laboratorio local sobre automatización durable con AutoKitteh y backend Temporal.

## Contenido
- `configs/.env.example`: variables mínimas de entorno
- `scripts/start-autokitteh-local.sh`: arranque local de servicios
- `scripts/run-workflow-sample.sh`: ejecución de workflow de prueba
- `workflows/event_ingest_workflow.py`: workflow Python de ejemplo
